python django 的一些小问题的解决办法

1.创建网站

django-admin startproject 项目名

2.创建一个app(创建完网站先进入网站的目录)

django-admin startapp app名

3.运行网站

python manage.py runserver 地址:端口

4.导入已经存在的数据库

python manage.py inspectdb >App路径/models.py

5.将models中的类生成migrations（适用于自己写model在导入数据库）

python manage.py makemigrations
将上面生成的结果导入数据库
python manage.py migrate

6.自定义自增主键

在自己要定义的自增主键中添加（primary_key=True）
